基础数学和逻辑

∀\forall∀表示“对于所有的”，或者 “对于每一个”。也可以读作“只要”。例如：∀\forall∀偶数n，nnn都能被2整除，翻译过来就是：对于所有的偶数n，都能被2整除。

∃\exist∃表示“存在”，例如∃ex\exist e^x∃ex，使得ddxex=ex\frac{d}{dx}e^x = e^xdxd​ex=ex，翻译就是，存在exe^xex，使得exe^xex的导数是exe^xex。

但是因为符号的增多，使得我们在读句子的时候需要非常小心。例如
∀x,∃y使得y>x∃y使得∀x，y>x\begin{aligned} &\forall x,\exist y 使得y > x\\ &\exist y 使得 \forall x， y>x \end{aligned}​∀x,∃y使得y>x∃y使得∀x，y>x​

第一个命题是：对于所有给定的xxx，存在yyy使得y>xy > xy>x。第二个命题是，存在yyy，使得对于所有的xxx都有y>xy > xy>x。如果x,yx,yx,y都是实数，对于第一句话是正确的，因为不管xxx多大，我们只要使得y=x+1y = x+1y=x+1就能使得第一句话永远正确。但是对于第二句话就不对了，因为无论yyy多大， 我们总能找到一个比他大的。这种过程是无穷无尽的，因此第二个命题就是假命题了。

举反例证明

在某些情况下，我们只需要举出一个反例，就能证明一些命题。例如：我说每个整数都是偶数，你只需要举出来3，这样子就能证明我是错的了。因此试着证明下面的例题

证明逆否命题

现在试着证明下面的例子：对于两个数xxx，yyy。仅当∀ϵ>0\forall \epsilon > 0∀ϵ>0,∣x−y∣<ϵ\vert x-y\vert < \epsilon∣x−y∣<ϵ时，有x=yx = yx=y

这个命题很经典了，就是极限的定义。命题断言，如果两个数非常的接近，这意味着我们可以选着任意小的距离ϵ\epsilonϵ，并且这两个数之间的距离比ϵ\epsilonϵ还要小，那么他们是相等的。

首先我们证明一般情况

1、x=yx = yx=y的时候，当然∀ϵ>0，∣x−y∣<ϵ\forall \epsilon > 0，\vert x-y\vert < \epsilon∀ϵ>0，∣x−y∣<ϵ

证明这个很简单，直觉上可以看出来，当x和y是一样的时候，他们之间的差必然是0.

2、∀ϵ>0，∣x−y∣<ϵ\forall \epsilon>0，\vert x-y\vert < \epsilon∀ϵ>0，∣x−y∣<ϵ等价于 x=yx = yx=y

这里讲的是，如果x,yx,yx,y之间的距离小于每一个正数，那么他们之间的距离必然是0.

现在我们得证明这个。即
x≠yx\neq yx=y 可以推出来存在一个ϵ>0\epsilon > 0ϵ>0使得∣x−y∣=∣z∣\vert x-y\vert = \vert z\vert∣x−y∣=∣z∣

如果x≠yx\neq yx=y，那么必然是xxx加上某个数会等于yyy。任何值的绝对值都是正数，因此我们就可以证明ϵ=∣z∣\epsilon=\vert z\vertϵ=∣z∣，那么自然有∣x−y∣=ϵ\vert x-y \vert = \epsilon∣x−y∣=ϵ，那么我们就证明了那个命题。他要么是第一种情况，要么是第二种，第二种明显不符合命题讲的，因此只有两个数一样的时候命题才能得证。

反证法

反证法跟上面是不一样的，反证法要证明的逆命题可以是成立也可以是不成立的，如果成立说明命题错误，如果不成立说明命题正确，而上面那个是两个方向都要证明出来。

怎么表示呢？我们要证明AAA能够推出BBB，我们假设AAA为真而BBB是假的，那么就会得出非常严重的错误，最终得到矛盾，例如0=10=10=1,5.35.35.3是整数类似的。因此当这种情况发生的时候，我们就证明了AAA如果为真，那么BBB不可能为假——否则数学大厦会崩塌。

例子

我们来证明2\sqrt{2}2​不是有理数。现在我们假设 2\sqrt{2}2​是一个有理数，并且证明这会导致一个很严重的错误。

在证明之前我们给出一些事实。

事实1：任何有理数都可以表示成mn\frac{m}{n}nm​的形式。并且如果mmm和nnn是偶数，我们可以上下同除2得到一个更简洁的形式。

事实2：对于某些整数a,ba,ba,b，只要a=2ba = 2ba=2b，那么aaa就是一个整数。因为可以被222整除。

事实3：如果aaa是奇数，那么a2a^2a2也是奇数。

现在我们开始证明，如果2\sqrt{2}2​是个有理数，那么一定可以表示成mn\frac{m}{n}nm​的形式。因此我们可以知道有两个数m,nm,nm,n可以组成如下形式
m2n2=2\frac{m^2}{n^2} = 2n2m2​=2

那么不难得出m2=2n2m^2 = 2n^2m2=2n2
基于事实2，我们可以知道mmm一定是一个偶数。并且利用事实3我们可以知道mmm如果是奇数，那么m2m^2m2也是个奇数，因此m一定是一个偶数。

现在我们把mmm写成2b2b2b表示一个偶数。那么m2=4b2m^2 = 4b^2m2=4b2，因此2n22n^22n2一定可以被4整除，那么n2n^2n2是一个偶数，但这出现问题了，我们一开始说了，m,nm,nm,n是互质的整数，即一个偶数另一个一定是奇数这种形式，我们得到了一个矛盾的命题。因此2\sqrt{2}2​不可能被表示成有理数的形式。

归纳法

归纳法就像推倒多米诺骨牌一样，如果我们把所有的牌摆好，推下第一张，那么剩下的都会倒下。这种情况适合证明需要证明无穷多种情形的情况。

他是这样子的：当定理在情形1成立的时候，在情形2也成立，在情形2成立的时候，在情形3下就一定成立。以此类推。所以可以先考虑第一种情形，然后假设在第n−1n-1n−1种情况下成立然后是nnn.

例子

我们试着找出前n个自然数之和的公式，即1+2+⋯+n1+2+\cdots +n1+2+⋯+n的求和公式，写成∑i=1ni\sum^n_{i=1}i∑i=1n​i，我们的大数学家，高斯找到了一个和，即n(n+1)2\frac{n(n+1)}{2}2n(n+1)​，现在我们带入几个值来验证。

基本情形1：当n=1n=1n=1的情况如下
∑i=11i=1(1+1)2=1\sum^1_{i=1} i = \frac{1(1+1)}{2} = 1i=1∑1​i=21(1+1)​=1
因此基本情形1成立。

归纳步骤：我们现在假设公式适用于n−1n-1n−1的情况，因此我们带入可得

∑i=1ni=∑i=1n−1+n=(n−1)n2+n=n2+n2=(n+1)n2\sum^{n}_{i=1}i = \sum^{n-1}_{i=1}+n = \frac{(n-1)n}{2}+n = \frac{n^2+n}{2} = \frac{(n+1)n}{2}i=1∑n​i=i=1∑n−1​+n=2(n−1)n​+n=2n2+n​=2(n+1)n​

现在我们证明完了，首先我们没有自己改变什么东西，都是按照归纳步骤来推导，即：如果适用于情形1，那么它也应该适用于情形2，如果它适应情形2，那么应该适应情形3.以此类推，就出来了。
