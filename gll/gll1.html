         ~~~~~~~~~         我们一般把观察或者实验的结果叫做事件，其中有可以分解 （复合） 事件和不可分解的 （简单） 事件。

例1、

         ~~~~~~~~~         三个球在三个盒子里面的分布，我们列出它的所有可能结果。
         ~~~~~~~~~         每一个排列都为一个简单事件，即一个样本点，事件A“某个盒子里面放了几个球”，第 1 到第 21 个排列的总体，我们说事件A是第1到第21个样本点所构成的集合。即3个球在三个盒子的全部排列。
         ~~~~~~~~~         事件B可以是：第一个盒子不空，那么第1、4、15和22-27这些样本点的集合。事件C为：A和B都发生。

生日问题

我们从这个著名的问题入手：房间里得有多少人才能保证其中至少两个人的生日在同一天的概率不小于50%？
我们当然假设在一年里面出生的概率是一样的，即1365\frac{1}{365}3651​，那么问题就很清楚了，如果房间里面每个人都与365天中的一天对应，那么有366人就足以满足这个问题。 这就是著名的 “狄利克雷鸽巢原理”(抽屉原理)
         ~~~~~~~~~         我们试着用穷举法来解决这问题。两个人的生日的排列有：3652=133225365^2 = 1332253652=133225，因为我们假设出生日期是相互独立的。而在这些排列中，两人生日相同的日子只有365天，那么我们算一下概率为：3653652≈0.27%\frac{365}{365^2} \approx 0.27\%3652365​≈0.27%
         ~~~~~~~~~         难题：随着人数的增多，我们的计算变得复杂，这使得我们很可能会重复计算，把一些本不该算进去的东西给算进去。那么我们提出一个方案——计算对立事件的概率
         ~~~~~~~~~         我们利用事件的互补关系得到，即两件事情必有一件发生。若A不发生，则B必发生的互补关系。即（1-p），例如我们计算n个人都互不相同的概率那么当n=1    ∃  p(A)=365365n = 1~~~~\exists ~~ p(A) = \frac{365}{365}n=1    ∃  p(A)=365365​ ，当n=2    ∃    365∗3643652n = 2 ~~~~\exists~~~~ \frac{365 * 364}{365^2}n=2    ∃    3652365∗364​ 第一个人的日期，第二个人不能用，所以要去掉。以此类推，n个人不同的生日为：
365∗(365−1)(365−2)⋅⋅⋅(365−n)365n\frac{365 * (365-1) ( 365 -2 ) ···(365-n)}{365^n}365n365∗(365−1)(365−2)⋅⋅⋅(365−n)​

那么计算相同的概率呢，就是去掉不同日期的概率即可得到。即
1−365∗(365−1)(365−2)⋅⋅⋅(365−n)365n1 - \frac{365 * (365-1) ( 365 -2 ) ···(365-n)}{365^n}1−365n365∗(365−1)(365−2)⋅⋅⋅(365−n)​
如果不用这种方法呢？

我们提一下重复计算，三个人的同一天生日的概率，这只能看成是一次成功，很多人会想成第一人和第二人相同日的概率，然后是第二人和第三人的概率，最后第一人和第三人相同日期生日的概率。很可能被错误的计算三次。用这种方法可以避免重复计算
n = 3 求两个人相同日期的概率。
p=365∗1∗3643653=39894548627125p = \frac{365 * 1 * 364}{365^3} = \frac{398945}{48627125}p=3653365∗1∗364​=48627125398945​
也可能会被写成：
p=365∗1∗3643653×3p = \frac{365 * 1 * 364}{365^3} \times 3p=3653365∗1∗364​×3
所以用如上互补方法可以解决这种重复计算。如果我们改变了一年的总天数，又得继续运算。但是如果是火星呢，火星一年有687天，但有趣的是，我们要得到答案只需要 313131个人。
          ~~~~~~~~~~          我们当然不可能碰到这种事情，但这种推广是很重要的。我们把问题转化成：一个随机试验有DDD种结果，并且每种结果出现的概率是相等的，那么需要进行多少次试验，才能使得至少两次结果相同的概率为50%50\%50%.

事件与事件之间的联系

连续样本空间

       ~~~~~~~       设一个函数是连续的并且解为满足某条件的集合，那么这是一个连续的样本空间，可以画两个圆圈去理解，圆圈里面的每个点都是样本点。[3]
我们给定一个样本空间Ω\OmegaΩ,用大写字母表示点在样本空间的事件。
       ~~~~~~~       1、不可能事件：不可能事件指的是不可能发生的事情，若事件A是不可能发生的事件，则用A=0A =0A=0表示不可能事件，在这里0是一个符号不是一个数值。
       ~~~~~~~       2、若事件A∈ΩA \in \OmegaA∈Ω，但是存在样本空间里面的一些点不属于事件AAA，那么我们把这些点叫做A’A’A’即AAA的补集，它包含所有不是AAA的点集
       ~~~~~~~       3、对于每一组事件A、B、C⋅⋅⋅A、B、C···A、B、C⋅⋅⋅可以把第二个新事件定义如下：第一个事件是由属于全部事件中给定的集合中的样本点组成的[4]，即A∩B∩C⋅⋅⋅A \cap B \cap C···A∩B∩C⋅⋅⋅这就是说，必须是全部给定的集合中的样本点同时发生，但我们知道一个样本点只能发生一次，所以同时发生就说明是交集。第二个事件是：由至少属于给定的各集合中之一的样本点的全体所组成也就是A∪B∪C⋅⋅⋅A \cup B\cup C···A∪B∪C⋅⋅⋅也就是说可以不依赖于其他的集合有发生的可能性，所以是并集。若AB=0AB = 0AB=0即没有公共点，我们说A&BA \And BA&B互斥
       ~~~~~~~       4、若A⊂B⇔B⊂AA \subset B \lrArr B \subset AA⊂B⇔B⊂A，我们讲AAA是被BBB包含的，也就是BBB的样本空间包含了整个AAA

离散样本空间

我们讲了连续的样本空间是什么样的，它像是一个圆圈，里面的东西（样本点）挨得很紧，但是离散不一样，他是散开的。不会挨紧。如果说连续的可以表示我们的年龄（因为每分每秒都得算上，不能用一个点代表一个刚刚好的年龄），那么离散好理解许多，它就像是一个课室的人一样，就这么多人，不会挨在一起。例如掷硬币，他只有两种情况，而我们前面说的例子，他可以有无穷多个（因为每分每秒都可以是一个样本点，又由于连续，所以有无穷多个可能）。那么到这里就很清晰了：离散样本空间就是有有限个的样本点的样本空间，而连续样本空间可以有无限个样本点。

从一个问题入手：投篮到几何级数。
          ~~~~~~~~~~          让我们想一下，有两个人一对一的展开投篮竞赛，设投进的概率A=pA = pA=p，B=qB = qB=q，失败的概率为：A=1−pA = 1-pA=1−p，B=1−qB = 1-qB=1−q，且AAA先投球，我们怎么知道AAA的获胜概率为多少呢？

准备知识：

          ~~~~~~~~~~          几何级数收敛公式：rrr为绝对值小于111的实数，
∑n=0∞rn=11−r\sum^{\infty}_{n=0}r^n = \frac{1}{1-r}n=0∑∞​rn=1−r1​

问题的开始:

我们设投篮的次数为nnn，我们计算第n次获胜后的概率，当n=1的时候，说明第一次投篮成功了，且概率为1.若n=2，那么第二次AAA的投篮是成功的，那么第一次是失败的，否则n不为2，那么第二次成功的概率为：(1−p)(1−q)p(1-p)(1-q)p(1−p)(1−q)p，从我们前面的解释来看：这里在说明第二次发生的概率，也就是第一次两个人失败的概率乘以成功的概率，这些事件同时发生，才能使得n=2n=2n=2这件事情发生，所以我们使得r=(1−p)(1−q)r= (1-p)(1-q)r=(1−p)(1−q)，如果AAA获胜，那么一定存在一个n，使得首次命中为第nnn次投篮。那么每次AAA投篮成功后的概率为：

∑n=0∞rnp=p∑n=0∞rn=p1−pr=(1−p)(1−q)\sum^{\infty}_{n=0}r^np = p\sum^{\infty}_{n=0}r^n = \frac{p}{1-p} \\ r = (1-p)(1-q)n=0∑∞​rnp=pn=0∑∞​rn=1−pp​r=(1−p)(1−q)
那么这里我们用无穷级数解决了这个问题。
