条件概率、独立性、贝叶斯定理

前言

如果什么事都是用各种可能的事件除一下得概率总是不对的。因为事件总不是无关联的独立状态。因为辩证唯物主义告诉我们：事物普遍具有联系性。这说明了事物并不总是独立存在的。我们举几个例子，从具体的现实事件来看。

例子

体育方面：
一、在一场足球比赛中，已知A球队在第75分钟以1−01-01−0暂时落后，那么他们获胜的概率有多大？
二、如果一个曲棍球队换了门将，那么该队继续取得第一次进攻的机会又有多少？

我们把这种含有前提条件（一般已经发生的事情）的事件叫做条件概率，记作Pr(A∣B)Pr(A|B)Pr(A∣B)读作：事件B发生的情况下A的概率是多少。注：本章会涉及到比较多的集合论知识，且这是最反直觉的一章，如果你了解过三门问题。

猜测条件概率公式

刚才我们给出了条件概率的表达Pr(A∣B)Pr(A|B)Pr(A∣B)
那么我们该如何找到一种有效的方式来计算呢？我们知道事件Pr(A∣B)Pr(A|B)Pr(A∣B)其实是可以看成是两个事件的，那么我们可以通过某种函数得到这个事实，于是设想我们可以像积分一样求出反导数来构造函数求这个概率的原函数，也就是做逆运算（不是真的求积分）。
我们设：
Pr(A∣B)=F(Pr(A),Pr(B))Pr(A|B) = F(Pr(A),Pr(B))Pr(A∣B)=F(Pr(A),Pr(B))
但这可能吗？对于论证我们始终要关注极端情况，我们让A=BA =BA=B或者可以让A=BcA = B^cA=Bc.也就是说我们可以让P(A)=P(B)P(A) = P(B)P(A)=P(B)或者P(A)=P(Bc)P(A) = P(B^c)P(A)=P(Bc)这只是分配概率的问题。现在我们开始为概率赋值。最简单的情况是事件的概率相等A=B=......A = B = ......A=B=......即
Pr(A)=Pr(B)=Pr(Bc)Pr(A) = Pr(B) = Pr(B^c)Pr(A)=Pr(B)=Pr(Bc)
根据"全概率公式"得到：
P(A)=1−P(Ac)P(A) = 1-P(A^c)P(A)=1−P(Ac)
同理可得，由于Pr(Bc)=Ω−Pr(B)Pr(B^c) = \Omega - Pr(B)Pr(Bc)=Ω−Pr(B)，所以我们知道
A⊂BcA \subset B^cA⊂Bc
而且由于
Pr(B)+Pr(Bc)=1，∴Pr(A,B,C)=12Pr(B) + Pr(B^c) = 1 ，\therefore Pr(A,B,C) = \frac{1}{2}Pr(B)+Pr(Bc)=1，∴Pr(A,B,C)=21​
因为除了这个事件，要不就是另一个事件。那么我们得到
1=Pr(B∣B)=F(12,12)=Pr(Bc∣B)=01 = Pr(B|B) = F (\frac{1}{2},\frac{1}{2}) = Pr(B^c|B) = 01=Pr(B∣B)=F(21​,21​)=Pr(Bc∣B)=0
如果我们想找这么一个函数，就得到了上面的谬论。因为我们假设了A、BA、BA、B事件是无关的，这是逻辑上的谬误，很常见。例如：你是动物，猪也是动物，所以你是猪。我们都知道这个不成立。我们把Pr(B∣B)Pr(B|B)Pr(B∣B)读作：当BBB事件发生的时候求BBB事件发生的概率。那么这个概率只能是1，因为已经发生了。Pr(BC∣B)Pr(B^C|B)Pr(BC∣B)读作：当BBB发生的时候，求BcB^cBc发生的概率。那么只能是0，因为BcB^cBc是BBB的补集，这说明和BBB无关，也就是说应该翻译成：当BBB发生的时候，求BBB不发生的概率！那么只能是0。所以条件概率一定和Pr(A)和Pr(B)Pr(A) 和Pr(B)Pr(A)和Pr(B)以外的事情有关。
到这里陷入了迷雾中，我们重新用数学的学习方法：不懂就多读几遍
条件概率求的是什么？是不是 当A发生的时候求B发生的概率 ？这意味着A、BA、BA、B都要发生，你想到了什么？对的，我们在第一章已经讲过了，同时发生的事情表现为两个集合（事件）中的交集。即Pr(A∩B)Pr(A\cap B)Pr(A∩B)
我们应该看下列表达式
Pr(A∣B)=G(P(A),P(B),P(A∩B))Pr(A|B) = G(P(A),P(B),P(A\cap B))Pr(A∣B)=G(P(A),P(B),P(A∩B))
记住：交集意味着两者同时发生
P(B∣B)=1P(Bc∣B)=00≤P(A∣B)≤1P(B|B) = 1\\ P(B^c|B) = 0\\ 0 \leq P(A|B) \leq 1P(B∣B)=1P(Bc∣B)=00≤P(A∣B)≤1
我们的概率公式是由这三个性质构成的.由于BBB是发生的事件，所以我们这个公式说明AAA在BBB里面占有多少，也就是比例。得到如下定义。
注：P(A∣B)P(A|B)P(A∣B)的样本空间在BBB里面，而P(AB)P(AB)P(AB)则在Ω\OmegaΩ里面
P(A∣B)=P(AB)P(B)，另一种写法=P(A∩B)P(B)P(A|B) = \frac{P(AB)}{P(B)} ，另一种写法=\frac{P(A\cap B)}{P(B)}P(A∣B)=P(B)P(AB)​，另一种写法=P(B)P(A∩B)​
由于我们考察的是BBB事件已经发生并且看AAA发生的概率，所以样本空间是在BBB上的，而不是全体样本空间。这是一样的，因为要完全除外发生的事情，来看没发生的事情！

P(A∩B)P(B)=0.40.65\frac{P(A\cap B)}{P(B)} = \frac{0.4}{0.65}P(B)P(A∩B)​=0.650.4​

乘法法则

由条件概率我们可以得到这么个公式：
Pr(A∩B)=Pr(A∣B)×P(B)\pmb{Pr(A\cap B) = Pr(A|B) \times P(B)}Pr(A∩B)=Pr(A∣B)×P(B)Pr(A∩B)=Pr(A∣B)×P(B)
这是什么意思呢，即——当A、BA、BA、B事件同时发生的时候的概率 = BBB事件发生后求AAA发生的概率 乘以 BBB事件的概率，且由于对称性我们知道P(A∩B)=Pr(B∣A)Pr(A)P(A\cap B) = Pr(B|A) Pr(A)P(A∩B)=Pr(B∣A)Pr(A)是成立的

独立性

上面的例子求Pr(A∩B)Pr(A\cap B)Pr(A∩B)是不是很麻烦，我说过概率的交集，可以用乘法表示，但这只能在A、BA、BA、B没有关系，也就是互斥的情况下实现。那么怎么样才算是独立事件呢？
我们想，刚才讲的条件概率：p(A∣B)=P(A)p(A|B) = P(A)p(A∣B)=P(A)，这说明B的事件他不会改变AAA事件的概率。那么他们是不相关的。
根据我们的乘法法则
P(A∩B)=P(A∣B)⋅P(B)P(A\cap B) = P(A|B) · P(B)P(A∩B)=P(A∣B)⋅P(B)
倘若事件独立则存在：P(A∣B)=P(A)P(A|B) = P(A)P(A∣B)=P(A)
由此我们得出：
P(A∩B)=P(A)⋅P(B)P(A\cap B) = P(A) ·P(B)P(A∩B)=P(A)⋅P(B)
定义如下：
若存在两个事件A、BA、BA、B，满足
Pr(A∩B)=Pr(A)⋅Pr(B)\pmb{Pr(A\cap B) = Pr(A)·Pr(B)}Pr(A∩B)=Pr(A)⋅Pr(B)Pr(A∩B)=Pr(A)⋅Pr(B)
那么事件是独立的

且我们有：事件A1....AnA_1 ....A_nA1​....An​是独立的，则有
P(A1∩A2∩.....An)=P(A1)⋅P(A2)⋅⋅⋅P(An)P(A_1 \cap A_2\cap .....A_n) = P(A_1) ·P(A_2)···P(A_n)P(A1​∩A2​∩.....An​)=P(A1​)⋅P(A2​)⋅⋅⋅P(An​)

贝叶斯公理

关于这个推导公式，我们先来理解一下为什么
P(A∩B)=P(B∩A)P(A\cap B) = P(B\cap A)P(A∩B)=P(B∩A)
我们上图吧。

我们看到，A∩BA\cap BA∩B是同时发生的，所以哪个先交都无所谓。因为它不是条件概率！
所以根据得到的这个等式，我们是不是也能证明
P(A∩B)=P(B∩A)=Pr(B∣A)⋅P(A)P(A\cap B) = P(B\cap A) = Pr(B|A) · P(A)P(A∩B)=P(B∩A)=Pr(B∣A)⋅P(A)
所以可以推出：
P(B∣A)⋅P(A)=P(A∣B)⋅P(B)P(B|A)·P(A) = P(A|B)·P(B)P(B∣A)⋅P(A)=P(A∣B)⋅P(B)
只要P(B)≠0P(B) \not ={0}P(B)=0我们就有
P(A∣B)=P(B∣A)⋅P(A)P(B)\pmb{P(A|B) = P(B|A)·\frac{P(A)}{P(B)}}P(A∣B)=P(B∣A)⋅P(B)P(A)​P(A∣B)=P(B∣A)⋅P(B)P(A)​
我们能推出这个定理的思路在于可交换性。我们用一个例子理解。

例子

我们设40%的人养狗40\% 的人养狗40%的人养狗，每年有20%20\%20%的养狗人会的流感，且每年有15%15\%15%的成年人的流感。那么已经得流感的某个人养狗的概率是多少？

根据贝叶斯定理我们可以得到
P(A∣B)=P(B∣A)⋅P(A)P(B)P(A|B) = P(B|A) ·\frac{P(A)}{P(B)}P(A∣B)=P(B∣A)⋅P(B)P(A)​、

P(A∣B)=0.2⋅0.40.15≈0.53P(A|B) = 0.2 · \frac{0.4}{0.15} ≈ 0.53P(A∣B)=0.2⋅0.150.4​≈0.53

那么贝叶斯概率是什么意思呢？
我们讲几个概念，这些概念是概率论非常重要的东西。
先验概率：如果在上面的例子，先验概率就是我们通过以往的经验分析出来的概率。例如一个骰子掷出点数的概率是平均的16\frac{1}{6}61​我们把观察到这个骰子的概率叫先验概率。先验概率其实并没有考虑其他的影响因素。

似然性，我们看贝叶斯公式的部分，我把它拆开分为：
P(A∣B)=P(B∣A)P(B)⋅P(A)P(A|B) =\frac{P(B|A)} {P(B)} · P(A)P(A∣B)=P(B)P(B∣A)​⋅P(A)
我们的贝叶斯公式可以这么理解：

新信息出现后的A概率 = A概率 乘以 新信息带来的调整。

也就是说我们的求解等于一个调整函数乘以一个事件，我们把等号右边的函数叫可能性函数，也就是看这个点落在交界范围的可能性有多大。 我们用流感的例子来讲解把，先列出我们已经有的条件。
A = 养狗，B = 得流感
(A｜B) = 得流感的人养狗
(B｜A) = 养狗的人得流感

P(A∣B)⏟得流感的人养狗的概率=P(B∣A)P(B)⏟如果得流感了，那么养狗的得流感概率大大提高 P(A)⏟养狗的人\underbrace{P(A|B)}_{得流感的人养狗的概率} = \underbrace{\frac{P(B|A)}{P(B)}}_{如果得流感了，那么养狗的得流感概率大大提高} ~\underbrace{P(A)}_{养狗的人}得流感的人养狗的概率P(A∣B)​​=如果得流感了，那么养狗的得流感概率大大提高P(B)P(B∣A)​​​ 养狗的人P(A)​​
现在我们知道了，P(A∣B)P(A|B)P(A∣B)就是求得流感的人在养狗的人里面占多少，而且是根据调整函数可以调整我们的概率的。
如果对于你来说很难理解概率的运算，那么我们可以用期望计数法来看。
假设一共有100个人（概率是比例，所以只要有样本就可以根据概率算大概值）

那么我们得到P(A∣B)=815≈53%P(A|B) = \frac{8}{15} ≈ 53\%P(A∣B)=158​≈53%

是一样的，如果你实在是不会用贝叶斯，你可以用这种方法去计算。

划分和全概率法则

我们看到贝叶斯概率是在条件概率上的展开，但要注意，P(A∣B)=P(B∣A)P(A|B) = P(B|A)P(A∣B)=P(B∣A)是很罕见的。因为即使我们可以交换运算次序，但是这种等同就像是和的平方根，和的平方根是不等于平方根的和的。
我们总是很难把贝叶斯定理用于实际，因为我们很难知道P(A)和P(B)P(A) 和P(B)P(A)和P(B)的概率，为什么？因为这两个事件总是或多或少的有交集。如果是独立事件则是另一个事情。由此，我们必须想办法计算P(A) and P(B)P(A) ~and~ P(B)P(A) and P(B)[1]
那么怎么办呢？我们知道概率的样本空间是什么？是具有某种性质的集合。也就是说，概率的本质就是可能发生事件的集合。
那么我们知道A∩B=∅A \cap B = \emptyA∩B=∅，那么这两个事件是互斥的。
注意到：
样本空间S的划分满足如下条件的可数个集合{A1A2...}(1、)存在i≠j,则Aj∩Ai=∅（2、）而全体Ai的并为整个样本空间——⋃iAi=S样本空间S的划分满足如下条件的可数个集合\{A_1A_2...\}\\ (1、)存在i \not ={j},则A_j\cap A_i =\empty \\ （2、）而全体A_i的并为整个样本空间——\bigcup_i A_i = S样本空间S的划分满足如下条件的可数个集合{A1​A2​...}(1、)存在i=j,则Aj​∩Ai​=∅（2、）而全体Ai​的并为整个样本空间——i⋃​Ai​=S
也就是说，我们把遥不可及分为很多个互不相交的集合
